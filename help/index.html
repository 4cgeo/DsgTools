Manual do usuário


Este plugin foi desenvolvido pela Diretoria de Serviço Geográfico (DSG) para ser possível executar a fase de aquisição cartográfica com o uso do QGIS. Este manual está dividido em submenus de acordo com as funcionalidades do plugin. Todas as funcionalidades presentes neste plugin são feitas de acordo com as Normas Técnicas do Sistema Cartográfico Nacional para a INDE que pode ser encontrada na página do Geoportal da DSG (http://www.geoportal.eb.mil.br/index.php/inde2 )
