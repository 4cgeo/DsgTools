Ferramentas de Camada Carregar por Categoria

Observação: Caso se deseje acessar um banco de dados já criado em vez de criar um novo através do DSG Tools, utilizar a ferramenta Adicionar camada PostGIS.

A funcionalidade de carregamento por categoria permite que camadas presentes no banco de dados sejam carregadas apenas com a definição da categoria. A figura abaixo mostra o diálogo de carregamento vazio.

Inicialmente deve ser feito o carregamento do banco desejado (PostGIS ou Spatialite). Tendo sido feito o carregamento a lista de categorias presentes no banco serão apresentadas na lista Carrega por Categoria presente no lado inferior esquerdo do diálogo.

Usando os botões >>, >, <, << é possível, respectivamente, carregar todas as categorias, carregar somente uma, descarregar uma e descarregar todas as categorias previamente carregadas.

O passo final se dá ao se selecionar a primitiva geométrica das categorias selecionadas que se quer carregar e se o usuário deseja carregar somente categorias que já possuem elementos (feições já existentes). Após selecionadas as categorias e opções desejadas, clicar em OK para realizar o carregamento.
