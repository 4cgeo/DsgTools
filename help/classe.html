Ferramentas de Camada Carregar por Classe

Observação: Caso se deseje acessar um banco de dados já criado em vez de criar um novo através do DSG Tools, utilizar a ferramenta Adicionar camada PostGIS.

A funcionalidade de carregamento por classe permite que uma única classe ou conjunto de classes sejam carregadas. As classes podem pertencer a diferentes categorias. A imagem abaixo mostra o diálogo de carregamento vazio.

O uso desta funcionalidade se inicia com o carregamento de um banco de dados previamente criado. Todas as classes presentes no banco são listadas no campo Selecione as Classes. Tendo sido selecionadas as classes desejadas é só clicar em OK que o carregamento seja efetuado.
