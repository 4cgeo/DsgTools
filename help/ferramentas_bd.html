Ferramentas de BD

Para se iniciar a aquisicação de dados com o uso do padrão da EDGV é necessário que se crie inicialmente um banco de dados.

O plugin permite que sejam criados bancos de dados em PostGIS e em Spatialite, além de permitir gerenciamento de usuários dos bancos PostGIS criados.

Observação: o nome do banco de dados, obrigatoriamente, deve ser em letras minúsculas.
