Requisitos Mínimos do Plugin DSG Tools

São necessários ao funcionamento adequado do plugin os seguintes programas:

  • Quantim GIS (2.6.1 ou superior)
  • PostgreSQL (9.3 ou superior)
  • PostGIS (2.0 ou superior)
  • SpatiaLite (4.2.0 ou superior)

Para um melhor funcionamento é recomendado o uso do Plugin Multiple Layer Selection. Ele permite a seleção em múltiplas camadas, o que facilita a criação de feições complexas.:

  • Multiple Layer Selection (0.3 ou superior)

Usuários de Linux

É necessário que se instale uma dependência para o funcionamento do plugin em Linux: a python-qt4-sql. Que pode ser instalada da seguinte forma:

sudo apt-get install python-qt4-sql.
