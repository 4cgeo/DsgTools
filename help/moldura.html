Ferramentas de Camada Criar Moldura

Observação: Em caso de se usar BD Spatialite, em vez de selecionar a conexão, é feita a seleção do arquivo SQLite.

Aqui é possível ver um vídeo mostrando como realizar a criação de molduras. Para maiores detalhes conceituais existem, abaixo do vídeo, os conceitos relacionados.

A funcionalidade de criar moldura permite a criação de uma área de trabalho para a carta que será trabalhada de acordo com o índice de nomenclatura informado pelo usuário (INOM). É possível ainda utilizar o campo MI (mapa índice) ou MIR (mapa índice reduzido). Da mesma forma é possível utilizar o campo MIR (mapa índice reduzido) para que automaticamente o campo INOM seja ajustado.

O banco de dados selecionado será utilizado para a criação da moldura. No caso do PostGIS a moldura fica presente na camada aux_moldura_a presente no schema público (public). No caso do Spatialite a moldura fica salva diretamente na camada aux_moldura_a.
